# Titanic-Data-Visualization-Dashboard

## Project Overview
This project is a mini data visualization dashboard built using Python, Matplotlib, and Seaborn.  
It analyzes the Titanic dataset and presents insights using different charts.

## Features
- Data Cleaning (handling missing values)
- Feature Engineering (FamilySize, AgeGroup)
- Visualizations:
  - Histogram (Age Distribution)
  - Bar Chart (Survival by Gender & Class)
  - Boxplot (Fare Distribution)
  - Scatter Plot (Age vs Fare)
  - Heatmap (Correlation)

## Key Insights
- Females had higher survival rates
- 1st class passengers survived more
- Higher fare passengers had better survival chances

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Titanic dataset (CSV file)
